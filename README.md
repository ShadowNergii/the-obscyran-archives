# The Obscyran Archives

The public-facing site for **The Obscyran Archives** — a living record of
Obscyra, the world of *PokéCyrus Online*, and the chronicle of Karel, its
first playable continent.

- **Live:** https://theobscyranarchives.net
- **Hosting:** Cloudflare Pages, auto-deploying from `main`
- **Source:** a single static `index.html` (no build step)
